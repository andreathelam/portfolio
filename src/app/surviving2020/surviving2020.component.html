OVERVIEW

Our game, Surviving 2020, is an Oregon-Trail esque parody game built in Twine, based around the theme of living during the time of the pandemic. It explores various topics from popular trends and political events from the year 2020. The goal of this game is to provide some comic relief as the player tries to survive 2020.

Building the Structure

Project Requirements

  • Access from a web browser
    • The game will be developed through Twine, which is a tool to create text-based web games. The end result of this project will be a playable, fully interactive game that users can access from any web browser on a device with internet connection.
  • Pandemic-themed
    • The focus of this game will be pandemic-themed. This includes the storylines, characters, and any design assets.
  • Single-player
    • The game will be created only for single-player.
  • Branching & multiple storylines
    • The type of this game will be interactive fiction. Users will read story content and interact by making decisions that will lead to the next event in the game. This is done through hyperlinks. Since there will be many options and paths for users to choose from, users will be able to explore multiple storylines with different events and alternate endings.
  • Original art and assets
    • All designs for the game will be created by the team. This will be done through Adobe Illustrator and Photoshop. By doing this, it creates an experience that is unique and different for the players to enjoy.

Creating the Art

All the art in the game was originally-created by our art team. The style we followed was pixel-themed to mimic the style from The Oregon Trail game. We used Aseprite, a pixel art tool, and Adobe Illustrator as the programs to design everything. During implementation, we added to the CSS stylesheet on Twine to develop the overall look of the final product.

Putting it All Together

The Mechanics

While navigating randomly-generated events, players must keep track of three main resources, Health, Happiness, and Money. These events are based on many of the shared experiences and frustrations that people have gone through during the pandemic, such as "losing your job" or "making banana bread" and will increase or decrease the player's resources. Each screen consists of descriptive text and hyperlink(s) that will lead the player to the next passage. Every action the player takes gives the player a chance of receiving Covid-19, and if players run out of their resources, they also are given a loss and will need to restart the game. To survive the game, players must last multiple rounds without emptying their resources until the vaccine has developed and they can receive it.

Takeaways

My team was unfamiliar with the structure of Twine, so we needed to learn and implement our game quickly while also encountering the challenges of working virtually. In this project course, we covered project management strategies, such as writing up a Work Breakdown Structure and following a Gantt Chart. I noticed a reoccurring problem was scope creep, of which we addressed through removing low-priority features or finding workarounds. The next steps for Surviving2020 would be to conduct user testing for quality assurance and revisions.
